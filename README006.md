# 006 - 배열과 흐름 제어문 활용

## 오늘 배운 것

- 배열을 활용하여 여러 개의 데이터를 저장할 수 있다.
- 조건문과 반복문을 활용하여 프로그램의 실행 흐름을 제어할 수 있다.

## 실습 소스 및 결과

- src/main/java/com/eomcs/lms/App.java 생성
- src/main/java/com/eomcs/lms/App2.java 생성
- src/main/java/com/eomcs/lms/App3.java 생성
